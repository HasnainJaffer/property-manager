LetroFlow — Logo package
=========================

Mark: "Home flow" — a house outline whose floor flows as a current.
Crystal brand system · indigo → cyan → mint gradient on a glass tile.

FILES
-----
letroflow-mark.svg              Primary icon, vector (use this wherever possible)
letroflow-lockup-dark.svg       Horizontal lockup for dark backgrounds (white "Letro")
letroflow-lockup-light.svg      Horizontal lockup for light backgrounds (ink "Letro")
png/letroflow-mark-*.png        Raster icon, transparent background
                                16,32,48,64,128,180,192,256,512,1024 px
                                (16/32 = favicon · 180 = Apple touch · 192 = Android)
png/letroflow-lockup-*.png      Raster horizontal lockups (1020×264)

COLOURS
-------
Indigo  #818cf8      Cyan  #22d3ee      Mint  #34d399
Wordmark on dark:  Letro #e7ecf3 · Flow indigo→cyan gradient
Wordmark on light: Letro #0f172a · Flow #4f46e5→#0891b2 gradient

USAGE
-----
• Prefer the SVG for screen and print — it stays crisp at any size.
• Keep clear space around the mark equal to ~25% of its height.
• Don't recolour the gradient, stretch, or add effects.
• The lockup SVGs reference the "Geist" font; if it isn't installed the
  text falls back to the system UI font. Outline the text before handing
  to print vendors if you need it pixel-locked.

Type family: Geist (https://vercel.com/font)
